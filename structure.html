STRUCTURE OF AN HTML DOCUMENT

An HTML document consists of specific tags that form the skeleton of the page. Here's a look at a basic HTML document structure


  1. <!DOCTYPE html>
  2. <html lang ="en">
  4. <head>
  5.     <meta charset="UTF-8" />
  6.     <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  7.     <title>Structure of HTML Document</title>
  8. </head>
  10. <body>
  11.     <!-- Main content of website -->
  12.     <h1>HTML Basic Structure</h1>
  13.     <p>This is how a basic HTML pagle is structured !</p>
  14. </body>
  16. </html>

The tags used in the block are explained in brief as follows :

  1. <!DOCTYPE HTML>: The <!DOCTYPE html> declaration is placed at the beginning of the document. It tells the browser that the document follows HTML5 standards, ensuring consistent rendering across browsers.

  2. <html> Tag: The <html> tag wraps the entire document, serving as the root element of an HTML page. It typically includes the lang attribute to specify the language of the content.

  3. <head> Section: The <head> section contains metadata, scripts, styles, and other information not displayed directly on the page but essential for functionality and SEO.

  4. <body> Section: The <body> section contains all the visible content of the web page, including text, images, videos, links, and more. This is where you"ll add the main elements to display on the page.


TAGS AND ELEMENTS FOR STRUCTURING AN HTML DOCUMENT

HTML tags are structural components enclosed in angle brackets. They are typically opened and closed with a forward slash (e.g., <h1></h1>). Some tags are self-closing, while others support attributes like width, height, and controls for defining properties or storing metadata.

There are generally two types of tags in HTML


  1. Paired Tags: These tags come in pairs i.e. they have both opening(< >) and closing(</ >) tags.

  2. Empty Tags: These tags are self-closing and do not require a closing tag.


HTML5

HTML 5 is the fifth and current version of HTML. It has improved the markup available for documents and has introduced application programming interfaces(API) and Document Object Model(DOM). It has introduced various new features like drag and drop, geo-location services
